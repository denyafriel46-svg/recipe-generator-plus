# üç≥ Recipe Generator Plus (Local LLM)

This is a simple **command-line application** that generates cooking recipes
using a **local AI model (LLaMA 3.1 8B)** running via **Ollama**.

The main goal of this project is to learn and demonstrate how to **integrate a local LLM**
into a real Python application **without using any cloud APIs**.

---

##  Highlights

- Uses **LLaMA 3.1 (8B)** locally
- No OpenAI or external API required
- Runs fully offline (after model is downloaded)
- Simple and clean Python structure
- Basic prompt engineering
- Suitable for **AI Engineer / Backend / ML portfolio**

---

##  What This App Does

With this app, users can:
- Input cooking ingredients
- Save ingredients to a file
- View, add, and remove ingredients
- Generate **Indonesian recipes** using a local LLM
- Interact through a simple CLI menu

---

##  Tech Stack

- Python 3.10  
- Ollama  
- LLaMA 3.1 (8B)  
- Command Line Interface (CLI)  
- Text file for simple storage
